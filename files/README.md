# Paradise Nursery Shopping Application

## Project Overview

Paradise Nursery is a front-end React shopping application that allows users to browse a curated collection of houseplants and manage a shopping cart.

## Features

- **Landing Page** – Background image, company description, and a "Get Started" button
- **Product Listing Page** – Browse 18+ houseplants grouped into 3 categories, with Add to Cart functionality
- **Shopping Cart Page** – View items, adjust quantities, see total cost, and proceed to checkout

## Tech Stack

- React 18 (Vite)
- Redux Toolkit (state management)
- CSS3
